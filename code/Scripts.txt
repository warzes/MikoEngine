Наверное лучше сделать свой, минималистичный
за пример доки по синтаксису взять https://dascript.org/doc/reference/language/lexical_structure.html

про переменные.
все переменные кроме под типов - умные указатели
Собственно переменные делятся на примитивные типы, и объекты

примитивные типа - это например числа и строки
пишутся так
int i = 32;

Объекты - все остальное. При этом они уже на умных указателях. Формат:
MyClass foo = new; // без аргументов
MyClass foo = new (10); //c аргументами.
MyClass foo2 = foo2; // указатель
MyClass foo2 = foo2.link; // указатель
MyClass foo2 = foo2.copy; // копия

команды delete нет. удалять вручную так:
foo = null;

при этом в билде все скрипты транслируются в с++ (а может сразу в си), и компилируются в бинарную либу (но можно и без этого - если нужно оставить открытым скрипты для модинга например)
